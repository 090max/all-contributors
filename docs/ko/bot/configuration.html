봇 설정

` .all-contributorsrc </ 0> JSON 파일을 업데이트하여 all-contributors bot 동작을 구성 할 수 있습니다. 기여자 목록을 생성하는 데 사용되는 데이터는 여기에 저장되며 @ all-contributors </ code>에서 목록을 생성하는 방법을 구성 할 수 있습니다.

CLI Configuration 과 유사한 봇 구성입니다.🤖

These are the keys you can specify:

Option Description Example/Default
files`

  "files": ["README.md"],
  "imageSize": 100,
  "contributorsPerLine": 7,

  "contributorTemplate": "<%= avatarBlock %><br /><%= contributions %>",
  "types": {
    "custom": {
      "symbol": "🔭",
      "description": "A custom contribution type.",
      "link": "[<%= symbol %>](<%= url %> \"<%= description %>\"),"

  },
  "contributors": []
