Star

All Contributors는 오픈 소스 프로젝트에 기여자를 인정하는 사양으로, 코드뿐 아니라 모든 기여물을 보상합니다.

기본적인 아이디어:

프로젝트 README (또는 프로젝트의 다른 유명한 공개 문서 페이지)를 사용하여 프로젝트 커뮤니티 멤버들의 공헌을 인정하세요.

많은 시간을 오픈 소스 프로젝트(코드나 다른분야)에 자유시간에 기여해주시 분들의 공헌에 대해 칭찬 받아야한다고 생각합니다.
