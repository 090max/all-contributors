번역은 crowdin를 통해 관리 됩니다. You can help contribute by heading to the crowdin project and suggesting translations. If you like, comment on this issue to let us know you're helping or if you have any queries!

Search

Search is by algolia, configuration is at https://github.com/algolia/docsearch-configs/blob/master/configs/all-contributors.json

Running the site locally:

All Contributors uses Docusaurus to maintain our documentation website. For more info visit the Docusaurus Website

To get started locally: - yarn install - yarn start - go to http://localhost:3000 (Refresh the page for your changes to be reflected)
