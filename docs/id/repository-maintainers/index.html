Catatan untuk Pemelihara Repositori

Kontributor harus diakui segera setelah kontribusi ditemukan untuk membantu memastikan upaya mereka tidak diabaikan ketika daftar diperbarui di kemudian hari. Ini bisa sulit untuk kontribusi yang bukan hasil dari komit ke repositori tetapi lakukan yang terbaik untuk meminimalkan waktu antara kontribusi dan memperbarui daftar.

Kiat pro: Jika proyek kamu dikonfigurasi untuk bot @all-contributors 🤖 cukup tulis komentar pada suatu isu atau pull request untuk mengakui kontribusi mereka. Sebagai contoh:

@all-contributors tolong add @tbenning untuk design ✨

Menjadi pemilik atau pengelola repositori tidak berarti kamu bertanggung jawab penuh untuk menyimpan daftar kontributor terkini. Kamu harus mendorong kontributor untuk menambahkan diri mereka ke daftar sebanyak mungkin. Ini bisa dalam bentuk komentar pada masalah, blog atau jawaban, atau melalui bentuk komunikasi yang lebih langsung jika sesuai.

Banyak kontributor mungkin tidak menyadari bahwa upaya mereka cukup untuk pengakuan dalam daftar kontributor. Ini mungkin karena mereka belum membaca atau memahami definisi kontributor seperti yang ditetapkan dalam spesifikasi ini, atau karena mereka merasa tidak cukup signifikan. Dalam kasus ini, Kamu harus tetap mendorong mereka untuk menambahkan diri mereka sendiri, tetapi mungkin perlu bagi kamu untuk menambahkan kontributor sendiri (meskipun itu ide yang baik untuk melakukannya dalam bentuk pull request untuk memastikan mereka ok saja untuk ditambahkan).

Pada akhirnya, tidak ada aturan yang keras dan cepat ketika kontribusi harus ditambahkan ke daftar, lakukan saja yang terbaik untuk bersikap adil dan untuk memastikan semua kontributor diakui.
