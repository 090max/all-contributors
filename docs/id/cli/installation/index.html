Pemasangan CLI

1. Pasang dari NPM

Modul ini didistribusikan melalui npm yang dibundel dengan node dan harus dipasang sebagai salah satu devDependencies proyek kamu:

yarn add --dev all-contributors-cli # atau npm

CLI berbasis node, jika kamu tidak ingin untuk menambah dependensi node gunakan bot @all-contributors 🤖

2. Inisialisasi Proyek

Inisialisasi proyeknya menggunakan init dan jawab beberapa pertanyaan

yarn all-contributors init # npm run

# Atau langsung jalankan bin
./node_modules/.bin/all-contributors init

3. Tambah beberapa kontributor

yarn all-contributors add jfmengels doc
yarn all-contributors generate

Untuk lebih lanjut tentang perintah, lihat penggunaan CLI

4. Perbarui dokumentasi Berkontribusi kamu

Pertimbangkan untuk memperbarui CONTRIBUTING.md kamu atau yang serupa dengan langkah-langkah tentang bagaimana kontributor kamu dapat menambahkan diri mereka sendiri. Kamu bahkan dapat menautkan ke dokumentasi penggunaan cli.

5. Opsional tambah skrip shortcut ke package.json kamu

Secara opsional kamu dapat menambah shortcut untuk perintahmu di package.json kolom scripts.

Sebagai contoh:

{
  "scripts": {
    "contributors:add": "all-contributors add",
    "contributors:generate": "all-contributors generate"
  }
}

Untuk membolehkan shortcut seperti

yarn contributors:add jfmengels doc
